```
✅ HTML Document Structure     — <!DOCTYPE>, <html>, <head>, <body>
✅ Text & Typography           — Headings, paragraphs, bold, italic
✅ Links & Navigation          — <a href>, internal/external links
✅ Images & Media              — <img>, <audio>, <video>, attributes
✅ Lists                       — Ordered, unordered, nested lists
✅ Forms & Inputs              — <form>, <input>, <button>, types
✅ Semantic HTML               — <header>, <nav>, <main>, <footer>
✅ Tables                      — <table>, <tr>, <td>, <th>
✅ Page Layout                 — Divs, sections, content organization
```

## 🛠️ How to Run Locally

```bash
# 1. Clone the repo
git clone https://github.com/Neo-0013/HTML-BASICS.git

# 2. Navigate into it
cd HTML-BASICS/freeCodeCamp

# 3. Open any project in your browser
open Bookstore.html
# or just double-click the .html file
```

> No dependencies. No frameworks. Pure HTML — just open in a browser and it works. 💪
